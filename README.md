# Vortrag über Werbung in Deutsch

Dieses Verzeichnis ist für einen Vortrag über Werbung in Deutsch gegründet worden.
Hier können alle Dateien für den Vortrag hochgeladen/heruntergeladen werden.
Jeder Teilnehmer dieses Projekt, bekommt ein Benutzerkonto mit E-Mail und Passwort.
Damit könnt ihr euch anmelden und Dateien verwalten. Es ist untersagt Dateien ohne
Absprache zu ändern. Dies kann zu einem Ausschluss/Verbannung dieses Projektes führen.
Bei Unstimmigkeiten bitte in der offiziellen WhatsApp-Gruppe dieses Projekts nach einer
Lösung suchen. Außerdem ist es unterlasssen in der WhatsApp-Gruppe zu spamen. Dies kann
ebenfalls zu einer Verbannung führen. Ich hoffe, dass dieses Projekt friedlich verläuft
und es zu keinem Ausschluss kommt. Doch wenn dies nötig ist, wird nicht daran gezweifelt
sondern radikal durchgeführt.
